# 声乐性格测试（VPT）版本更新公告

## v1.0.0 — 首次发布

### 关于

声乐性格测试（Vocal Personality Test，简称 VPT）是一款基于 MBTI 四维度划分逻辑、专为中老年声乐爱好者打造的在线性格测试应用。通过 20 道贴近演唱场景的测试题，帮助你发现自己的专属演唱人格，并获得个性化的经典老歌推荐。

### 核心功能

- **20 道声乐性格测试题** — 围绕演唱动力源、演唱表达偏好、演唱决策方式、演唱习惯四大维度，每维度 5 道题，贴合中老年用户的演唱与生活场景
- **8 种声乐人格类型** — 合唱领唱者、温情传唱者、经典创新者、韵味歌者、精益演唱者、治愈传唱者、小众怀旧者、情怀吟唱者
- **性格徽标展示** — 以轻奢风纹章式徽标呈现测试结果，四维度倾向一目了然
- **个性化歌曲推荐** — 根据你的声乐人格，推荐 3 首最适配的经典老歌，附推荐理由
- **图片分享** — 一键生成精美的结果卡片图片，可保存或分享给歌友

### 设计特色

- **轻奢风格** — 象牙色暖底搭配金色点缀，优雅克制
- **适老化设计** — 大字号、大按钮、大触摸区域、高对比度，中老年用户友好
- **响应式布局** — 适配手机与桌面端，随时随地畅享测试

### 技术栈

- Next.js 16 + React 19 + TypeScript 5
- Tailwind CSS 4 + shadcn/ui
- html2canvas 图片导出
